Getting started

Requirements:

  • Chrome > 89 (enabled by default)
  • Chrome < 89: Go to chrome://flags and enable "Experimental Web Platform features", then restart your browser

Note: For linux, you need to ensure udev is setup with the correct device permissions for the hidraw driver.
